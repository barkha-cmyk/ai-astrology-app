═══════════════════════════════════════════════════════════════
  HOW TO GENERATE API KEY - SIMPLE STEPS
═══════════════════════════════════════════════════════════════

RECOMMENDED: Use OpenAI (Easier & More Reliable)
═══════════════════════════════════════════════════════════════

STEP 1: CREATE ACCOUNT
───────────────────────────────────────────────────────────────
1. Visit: https://platform.openai.com/signup
2. Click "Sign up"
3. Enter email and password
4. Verify your email

STEP 2: ADD PAYMENT METHOD
───────────────────────────────────────────────────────────────
1. Visit: https://platform.openai.com/account/billing
2. Click "Add payment method"
3. Enter credit card details
4. Save payment method

STEP 3: ADD CREDITS
───────────────────────────────────────────────────────────────
1. On billing page, click "Add credits" or "Add funds"
2. Enter amount: $5 (minimum recommended)
3. Complete payment

STEP 4: GENERATE API KEY
───────────────────────────────────────────────────────────────
1. Visit: https://platform.openai.com/api-keys
2. Click "Create new secret key" (top right button)
3. Enter name: "AI Astrologer App"
4. Click "Create secret key"
5. COPY THE KEY IMMEDIATELY (you won't see it again!)
   It looks like: sk-proj-abc123xyz789...

STEP 5: ADD TO YOUR PROJECT
───────────────────────────────────────────────────────────────
1. Open .env.local file in your project
2. Find this line: OPENAI_API_KEY=
3. Replace with: OPENAI_API_KEY=sk-proj-YOUR_ACTUAL_KEY_HERE
4. Save the file

STEP 6: RESTART SERVER
───────────────────────────────────────────────────────────────
1. Stop server (Ctrl+C in terminal)
2. Run: npm run dev:8080
3. Wait 30 seconds
4. Test at: http://localhost:8080/chat

═══════════════════════════════════════════════════════════════
ALTERNATIVE: Use OpenRouter
═══════════════════════════════════════════════════════════════

STEP 1: CREATE ACCOUNT
───────────────────────────────────────────────────────────────
1. Visit: https://openrouter.ai/
2. Click "Sign up"
3. Enter email and password
4. Verify your email

STEP 2: ADD CREDITS (REQUIRED!)
───────────────────────────────────────────────────────────────
1. Visit: https://openrouter.ai/settings/credits
2. Click "Add Credits" or "Top Up"
3. Enter amount: $5 (minimum required)
4. Complete payment

STEP 3: GENERATE API KEY
───────────────────────────────────────────────────────────────
1. Visit: https://openrouter.ai/keys
2. Click "Create Key" or "New Key"
3. Enter name: "AI Astrologer"
4. Click "Create"
5. COPY THE KEY (starts with sk-or-v1-...)

STEP 4: ADD TO YOUR PROJECT
───────────────────────────────────────────────────────────────
1. Open .env.local file
2. Find: OPENROUTER_API_KEY=...
3. Replace with: OPENROUTER_API_KEY=sk-or-v1-YOUR_NEW_KEY_HERE
4. Save the file

STEP 5: RESTART SERVER
───────────────────────────────────────────────────────────────
1. Stop server (Ctrl+C)
2. Run: npm run dev:8080
3. Wait 30 seconds
4. Test at: http://localhost:8080/chat

IMPORTANT NOTES
═══════════════════════════════════════════════════════════════

✓ Never share your API key publicly
✓ Save the key securely (password manager)
✓ Start with $5 to test, then add more
✓ Check server logs to verify it's working
